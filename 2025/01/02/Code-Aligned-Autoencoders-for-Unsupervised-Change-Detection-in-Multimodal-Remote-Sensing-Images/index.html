Code-Aligned Autoencoders for Unsupervised Change Detection in Multimodal Remote Sensing Images

  • 题目：编码对齐的自编码器用于多模态遥感图像的无监督变化检测

  • 作者：Luigi Tommaso Luppino，Mads Adrian Hansen. etc

  • 发表时间及期刊：2024 IEEE Transactions on Neural Networks and Learning Systems

论文结构

  • Abstract

    介绍论文模型，结果在数据集中被证实有效性

  • Introduction

    异质变化检测常见的解决方法是应用高度非线性的变换，将数据映射到相同的域或空间进行比较。但是这需要迭代微调和可靠的初始结果。

    • 条件生成对抗网络 (cGAN): 一种基于生成对抗网络的图像翻译模型，它需要配对的图像进行训练，例如包含相同对象的图像，但风格或传感器模式不同。
    • cycleGAN: 一种改进的 cGAN 模型，它不需要配对的图像进行训练，而是通过循环一致性来保证翻译的准确性。

    cGAN和cycleGAN可能成功地对齐转换数据和真实数据的分布，但它们也存在固有的缺点：它们依赖于大型训练集，生成器和鉴别器的迭代训练必须谨慎平衡，训练容易发生模式崩溃，由于损失函数的振荡和不稳定行为，很难找到超参数的合理值。

    方法：两个自编码器的编码层对齐，视为一个共同的潜在空间，分析输入图像的局部特征，无监督地指导编码层的对齐过程。

    这一原理的实现受到了Kampffmeyer等人[41]和Bianchi等人[42]的深度核化自编码器的启发，其中强制两个数据点产生的编码的内积与它们预计算的亲和度相匹配。

  • 贡献

    1. 提出了一个简单但有效的损失项来无监督地对齐两个自编码器的潜在空间。
    2. 利用该损失项实现了一个深度神经网络
    3. 开放了源码，方便与其他TensorFlow 2.0框架的方法比较。地址在：https://github.com/llu025/Heterogeneous_CD

  • Method

    如果没有外部干预，两个自编码器（一个处理X域数据，另一个处理Y域数据）学习的代码分布（ZX和ZY）通常不会相似。图1(a)提供了一个视觉示例来说明这一点。

    为了使两个自编码器的代码分布对齐，研究者引入了特定的损失项。这些损失项的目的是在两个方面强制对齐：一是代码的分布本身，二是代码在潜在空间中表示的地物覆盖的位置。

    如果ZX和ZY中的代码分布成功对齐，这意味着两个自编码器在潜在空间中找到了共同的表示。这样，编码器可以与解码器级联使用。

    • 与相邻的解码器级联：可以将潜在空间的代码映射回它们的原始域，即X域的编码器与X域的解码器级联，Y域的编码器与Y域的解码器级联。
    • 与相反的解码器级联：可以将数据从一个域映射到另一个域，即X域的编码器与Y域的解码器级联，或者Y域的编码器与X域的解码器级联。这实现了跨域的数据变换，即所追求的变换。

  • Dataset

    Texas，California，Italy，France

  • Results

摘要核心

  1. 卷积自动编码器在双时态卫星图像多模态变化检测中近年来十分流行。

  2. 主要挑战是减少变化像素对转换函数学习的影响来对齐编码空间

  3. 许多现有方法训练网络利用到变化区域的监督信息，这种信息不总是可用。

  4. 利用输入域的特定领域亲和矩阵提取关系像素信息来强制对齐编码空间，减少变化像素对学习目标的影响（权重）

    • 变化像素通常在亲和矩阵中表现出与未变化区域不同的关系，因此它们不会主导编码空间的对齐过程。
    • 通过强调未变化区域的关系，模型可以更好地学习到这些区域的内在特征，而不是变化像素的异常特征。
    • 这种方法使得自编码器更专注于学习未变化区域的稳定特征，从而在变化检测任务中减少了对变化像素的依赖。

  5. 利用跨领域的像素对亲和度无监督地推出变化先验，再强制输入域中有相似亲和关系的像素在编码空间也有相关性来实现编码空间对齐；该方法在四个数据集上证实了有效性。

BiFA: Remote Sensing Image Change Detection With Bitemporal Feature Alignment

  • 题目：BiFA: 基于双时相特征对齐的遥感图像变化检测
  • 作者：Haotian Zhang
  • 发表时间及期刊：2024 IEEE Transactions on Geoscience and Remote Sensing

论文结构

  • Abstract

    介绍论文模型，在六个数据集中得到最优结果

  • Introduction

    现有方法存在的问题：

    1. 双时相交互（BI）不足：在特征提取阶段，现有方法未能有效利用两个时间点图像之间的交互信息，可能导致无法有效区分实际变化和由光照、季节等因素引起的变化。
    2. 空间对齐局限：现有方法实现注意力机制来实现空间对齐，但没有专注于预测像素级的偏移量。
    3. 多尺度对齐局限：现有方法要么牺牲精度（双线性插值），要么引入额外参数（如反卷积）。使得同时保持模型轻量化和实现精确多尺度特征对齐具有挑战。

  • Method

    BiFA由三个关键模块组成，双时相交互（BI）模块、基于微分流场的对齐模块（ADFF）和隐式神经对齐解码器（IND）

    针对上述方法的局限性，我们提出了双时域特征对齐（BiFA），通过结合TA（信道和空间）和多尺度对齐来解决BiFA的不足。该网络由三个关键模块组成，即BI模块、基于微分流场的对齐模块（ADFF）和隐式神经对齐解码器（IND）。对于TA，BI模块的开发是为了解决特征提取阶段需要深度双时态特征交互的问题。将特征交互纳入骨干网有助于实现双时图像特征的信道对齐。在我们的直觉中，在另一个时间特征的指导下，它可以抑制不相关因素的干扰，如光照和季节变化，并提取出更稳健的特征。ADFF模块用于解决由于不同视角引起的配准不足的问题，并明确估计双时图像之间的像素级偏移，以实现双时图像的空间对齐，从而减轻由于偏移引起的错误检测。对于多尺度对齐，IND旨在获取坐标空间内图像的连续表示，用于解决多尺度特征对齐问题，并通过轻量级方法实现多尺度特征之间的精确对齐，以提高CD的性能

    TA块有BI模块和ADFF组成，实现通道级和空间级对齐，并产生多级差分特征。然后送到IND，轻量级地对齐，将差异特征图转换为连续的特征图并生成准确的预测图。

    通过线性映射将特征投影到不同的子空间，实现通道维度的对齐。

    微分流场：用来描述在不同时间点捕获图像特征之间的空间位移。

    用FlowMix模块增强F1、F2特征表示。通过一个卷积层生成微分流场，存储每个位置的偏移量。

    将原始坐标p1与微分流场Δf相加，得到对齐后的坐标p̂。由于位移产生的缺失值，使用可微分的双线性采样技术来填充，得到对齐后的特征F′warp。

    通过对齐后的特征F′warp与原始特征F2进行绝对值相减，得到时空差分特征Fdiff。

    IND：对于一个查询坐标（红点），我们从每一层中获得最近的隐藏代码 $z_{i}^{*}$ （绿点）、相对坐标 $\Delta x_{l}$ 和相对坐标的位置编码 $\beta(\Delta x_{l})$ 。然后，将它们连接起来并输入解码函数 $f_{\theta}$ ，该函数由三层MLP组成。$f_{\theta}$ 解码每个阶段的特征，并同时模拟各个阶段之间的互连。这样，就可以以一种轻量级的方式对多级微分地图进行对齐，从而获得高质量的预测图。

  • Dataset

    WHU-CD、LEVIR-CD、LEVIR+-CD、SYSU-CD、DSIFN-CD、CLCD

  • Results

摘要核心

  1. 目前的基于深度学习的变化检测方法在时间（通道和空间）和多尺度对齐方面有所不足
  2. 提出了双时相特征对齐BiFA模型，减少无关因素的影响，轻量级地生成精确的变换检测图。
  3. 通过在时间、空间、多尺度上对齐，模型在六个数据集上达到最优，并在跨分辨率变化检测中显示出很强的鲁棒性。
  4. 源码：https://github.com/zmoka-zht/BiFA
